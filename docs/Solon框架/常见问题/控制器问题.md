# 控制器问题

## 2.1 控制器方法无法被访问

**问题描述**：访问控制器方法时返回 404 错误。

**解决方法**：
- 检查控制器类是否添加了 `@Controller` 注解。
- 检查方法是否添加了 `@Mapping` 注解并指定了正确的路径。
- 检查路径是否包含上下文路径（如果配置了的话）。
- 检查 HTTP 请求方法是否匹配（默认支持 GET 和 POST）。

## 2.2 路径参数无法正确获取

**问题描述**：控制器方法无法正确获取路径参数的值。

**解决方法**：
- 确保路径参数名称与方法参数名称一致，或者使用 `@Param` 注解指定映射关系：
  ```java
  @Mapping("/user/{id}")
  public String getUser(@Param("id") int userId) {
      return "User ID: " + userId;
  }
  ```

## 2.3 请求体参数无法正确绑定

**问题描述**：POST 请求时，请求体参数无法正确绑定到方法参数。

**解决方法**：
- 确保请求头 `Content-Type` 设置为 `application/json`。
- 确保参数类型是一个 Java Bean 类，并且具有默认构造方法和 setter 方法。
- 检查 JSON 格式是否正确。
