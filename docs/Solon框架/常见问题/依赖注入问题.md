# 依赖注入问题

## 3.1 依赖注入失败

**问题描述**：启动时出现依赖注入失败的错误。

**解决方法**：
- 检查被注入的类是否添加了 `@Component`、`@Service`、`@Repository` 等注解。
- 检查注入点是否添加了 `@Inject` 注解。
- 检查是否存在多个实现类，导致依赖注入歧义，此时需要使用 `@Named` 注解指定具体实现：
  ```java
  @Inject
  @Named("helloServiceImpl")
  HelloService helloService;
  ```

## 3.2 循环依赖问题

**问题描述**：启动时出现循环依赖的错误。

**解决方法**：
- 重构代码，打破循环依赖关系。
- 使用 `@Lazy` 注解延迟加载依赖：
  ```java
  @Inject
  @Lazy
  private ServiceA serviceA;
  ```
