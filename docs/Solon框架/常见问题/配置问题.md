# 配置问题

## 5.1 配置项无法读取

**问题描述**：使用 `@Inject("${key}")` 注解无法读取配置项的值。

**解决方法**：
- 检查配置文件中是否存在对应的配置项。
- 检查配置项的路径是否正确（使用点分隔符）。
- 检查配置项的值类型是否与注入的变量类型匹配。

## 5.2 多环境配置问题

**问题描述**：需要在不同环境（开发、测试、生产）使用不同的配置。

**解决方法**：
- 使用多环境配置文件，如 `app-dev.yml`、`app-test.yml`、`app-prod.yml`。
- 在启动时指定环境：
  ```bash
  java -jar app.jar --solon.env=dev
  ```
