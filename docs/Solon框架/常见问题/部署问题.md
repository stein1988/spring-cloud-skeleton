# 部署问题

## 7.1 打包后的 jar 包过大

**问题描述**：Maven 打包后的 jar 包体积过大，影响部署速度。

**解决方法**：
- 使用 `maven-shade-plugin` 或 `spring-boot-maven-plugin` 进行打包优化。
- 排除不必要的依赖。
- 考虑使用分层构建减少镜像大小。

## 7.2 容器化部署问题

**问题描述**：在 Docker 容器中运行应用时出现问题。

**解决方法**：
- 确保 Dockerfile 配置正确：
  ```dockerfile
  FROM openjdk:11-jre-slim
  COPY target/solon-demo-1.0.0.jar /app/app.jar
  WORKDIR /app
  EXPOSE 8080
  CMD ["java", "-jar", "app.jar"]
  ```
- 检查容器网络配置是否正确。
- 确保环境变量设置正确。
