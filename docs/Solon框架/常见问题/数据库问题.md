# 数据库问题

## 4.1 数据库连接失败

**问题描述**：应用无法连接到数据库，出现连接超时或认证失败的错误。

**解决方法**：
- 检查数据库连接配置是否正确（URL、用户名、密码）。
- 检查数据库服务是否正常运行。
- 检查网络连接是否正常，防火墙是否允许连接。

## 4.2 SQL 执行错误

**问题描述**：执行 SQL 语句时出现语法错误或其他数据库错误。

**解决方法**：
- 检查 SQL 语句是否正确。
- 检查参数类型是否匹配。
- 检查数据库表结构是否与代码中的实体类一致。

## 4.3 事务不生效

**问题描述**：使用 `@Transactional` 注解标记的方法，事务不生效。

**解决方法**：
- 确保方法是 public 的（事务注解只对 public 方法生效）。
- 确保方法不是在同一个类中调用的（Spring AOP 的限制）。
- 检查是否添加了事务管理器配置。
