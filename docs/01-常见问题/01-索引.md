# 常见问题索引

本文档提供了项目中常见问题的索引，帮助开发人员快速定位和解决问题。

## 问题列表

### 1. easy-query 插入数据无效

**问题描述**：使用 easy-query 框架插入数据后，在数据库中找不到插入的数据，或者查询不到插入的数据，但 easy-query 的其他查询操作是正常的。

**解决方案**：请参考文档 [02-easy-query插入数据无效.md](02-easy-query插入数据无效.md)。

### 2. 异常处理系统相关问题

**问题描述**：项目中出现异常但没有被正确捕获和处理，导致系统行为异常。

**解决方案**：请参考文档 [04-Spring框架/02-异常处理/01-全局异常处理.md](../04-Spring框架/02-异常处理/01-全局异常处理.md)。

### 3. 数据库时间戳字段处理

**问题描述**：PostgreSQL 的 timestamptz 类型与 Java 类型的映射问题，特别是时区处理。

**解决方案**：请参考文档 [03-数据库规范/03-时间戳字段.md](../03-数据库规范/03-时间戳字段.md)。

### 4. 模块化开发相关问题

**问题描述**：在模块化开发过程中，遇到模块依赖、组件扫描、自动配置等问题。

**解决方案**：请参考文档 [04-Spring框架/01-模块化开发/01-基础模块组件的依赖注入.md](../04-Spring框架/01-模块化开发/01-基础模块组件的依赖注入.md)。

## 如何贡献

如果您在项目中遇到了新的常见问题，欢迎您贡献解决方案：

1. 在 `docs\01-常见问题` 文件夹中创建新的文档文件，命名格式为 `XX-问题描述.md`，其中 XX 是序号。
2. 在文档中详细描述问题、排查步骤和解决方案。
3. 更新本文档，在问题列表中添加新的问题条目。

## 注意事项

- 请确保文档内容准确、清晰、完整，便于其他开发人员理解和使用。
- 请使用 Markdown 格式编写文档，保持格式一致。
- 请在文档中提供具体的代码示例和配置示例，以便其他开发人员参考。
