# Easy-Query 简介

## 什么是 Easy-Query

Easy-Query 是一款功能强大的 Java ORM 框架，旨在提供简单、高效、灵活的数据库操作能力。它支持多种数据库，包括 MySQL、PostgreSQL、Oracle、SQL Server 等，提供了丰富的查询 API 和便捷的实体映射功能。

## 官方文档

- 官方网站：[https://www.easy-query.com/](https://www.easy-query.com/)

## Easy-Query 与 MyBatis 对比

### 编程模型

- **Easy-Query**：采用链式 API 风格，支持类型安全的查询构建，无需编写 XML 文件。
- **MyBatis**：需要编写 XML 映射文件或使用注解来定义 SQL 语句。

### 性能

- **Easy-Query**：内置了多种性能优化策略，如查询缓存、预编译 SQL 等，性能表现优异。
- **MyBatis**：性能良好，但需要手动优化 SQL 语句和缓存策略。

### 功能特性

- **Easy-Query**：
  - 支持链式查询 API
  - 支持实体代理（EntityProxy）
  - 支持复杂查询条件
  - 支持批量操作
  - 支持事务管理
  - 支持多数据源

- **MyBatis**：
  - 支持 XML 配置
  - 支持动态 SQL
  - 支持缓存
  - 支持插件扩展

## 应用

在本项目中，我们使用 Easy-Query 作为 ORM 框架，结合 Solon 框架构建应用。
